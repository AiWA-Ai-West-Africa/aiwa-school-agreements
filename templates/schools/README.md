# Templates — Schools

This folder holds school-specific template adaptations — versions of standard AIWA
templates that have been pre-populated or customised for a specific partner school.

## Naming Convention

```
[category]-[SCHOOL_ID]-template-v[VERSION].md
```

Example: `contract-BJL-PRI-001-template-v1.0.md`
