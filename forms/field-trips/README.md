# Field Trip Permission and Travel Forms

This folder holds AIWA's permission and travel agreement forms for on-location and off-campus activities.

## Contents

| File | Description |
|---|---|
| `form-field-trip-permission-v1.0.md` | Parent/guardian permission slip and travel agreement for on-location media production sessions and off-campus activities |

## When to Use

A Field Trip Permission and Travel Agreement must be collected from a student's parent or guardian before the student participates in any activity that takes place outside the school premises. This includes:

- On-location media production sessions
- Community visits and oral history recording sessions
- Author visits or events at external venues
- Any off-campus AIWA activity
