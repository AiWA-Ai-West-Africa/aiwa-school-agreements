# Contracts — Archive

This folder contains superseded, expired, or withdrawn contract templates and agreements.

## Purpose

Documents are archived here when:
- A new approved version of a template has been released
- A contract has reached the end of its term
- A contract has been formally terminated or withdrawn

Archived documents must not be deleted. They form part of the institutional record.

## Naming Convention

Archived files retain their original name with the archival suffix:
```
[original-filename]-archived-[YYYY-MM-DD].[ext]
```
