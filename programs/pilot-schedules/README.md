# Programs — Pilot Schedules

This folder holds school-specific programme timetables and academic year schedules.

## Contents

Add pilot schedule documents here. Files should be named:

```
schedule-[SCHOOL_ID]-[ACADEMIC_YEAR_SLUG]-v[VERSION].md
```

Where `[ACADEMIC_YEAR_SLUG]` is the **file-safe** academic year in hyphenated format (e.g., `2025-2026`).
This differs from the `[ACADEMIC_YEAR]` placeholder used *inside* documents, which uses a slash (e.g., `2025/2026`).
Slashes are not safe in file names and must not be used.

Example: `schedule-BJL-PRI-001-2025-2026-v1.0.md`
